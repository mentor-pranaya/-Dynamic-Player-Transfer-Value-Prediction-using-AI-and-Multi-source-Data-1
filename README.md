TransferIQ — Dynamic Player Transfer Value Prediction

AI-powered system that predicts football player transfer values using performance statistics, injury data, sentiment analysis, and machine learning models (LSTM + XGBoost).

🔥 Project Overview

TransferIQ integrates multi-source data to estimate real-time player transfer value:

Player performance datasets

Market value history

Twitter sentiment analysis

Injury records

Time-series modeling (LSTM)

XGBoost ensemble regression

The project includes a fully functional Streamlit UI for demonstrating each stage of the pipeline.
