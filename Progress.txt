For Week1: This week, I worked on data collection and sentiment analysis using a Kaggle dataset on FIFA World Cup 2022 tweets. I cleaned the data using Pandas, analyzed sentiments with TextBlob, and visualized the results using Matplotlib. Most tweets showed positive sentiment, reflecting public excitement about the event. Through this task, I learned how to preprocess text data, perform sentiment analysis, and generate insights using Python 3.11.

For Week2: This week, I completed the data cleaning and preprocessing stage of my AI internship project. I handled missing and duplicate data, encoded categorical features, scaled numerical values, and saved the processed dataset (transfermarkt_preprocessed.csv). This prepared the data for feature selection and model training in Week 3.

For Week3: During Week 3, I created advanced features like player experience and value-trend ratios, integrated sentiment scores from football tweets, and trained a Random Forest Regressor to predict player market values. The model achieved strong performance, and the analysis showed that position, club, and public sentiment significantly affect player valuation.
