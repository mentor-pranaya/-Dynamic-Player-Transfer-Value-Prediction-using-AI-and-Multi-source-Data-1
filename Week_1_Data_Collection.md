Week 1 – Data Collection 
Project: Football Player Analytics

1. Player Performance Data
- Source: StatsBomb Open Data
- Link: https://github.com/statsbomb/open-data
- Description: I downloaded player and match data from StatsBomb. This includes stats like goals, assists, passes, shots, and minutes played. This will help analyze how players perform in different matches and seasons.

2. Market Value Data
- Source: Transfermarkt
- Link: https://www.transfermarkt.com/spieler-statistik/wertvollstespieler/marktwertetop
- Description: I collected player market values from Transfermarkt. The data has player names, clubs, positions, ages, and market values. This will be useful to compare player performance and their market price.

3. Sentiment Data
- Source: Twitter API
- Description: I used the Twitter API to get tweets about selected players. These tweets will later be used for sentiment analysis to see how fans react to player performance or news.

4. Injury Data
- Source: Football Datasets by Salimt
- Link: https://github.com/salimt/football-datasets/tree/main/datalake/transfermarkt/player_injuries
- Description: I downloaded injury data of players. It contains injury types, start and end dates, days missed, and games missed. This data will help to check how injuries affect performance and market value.
